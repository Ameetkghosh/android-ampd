    android Music Player Daemon (aMPD)

    Project maintained by tguillem Hosted on GitHub Pages — Theme by mattgraham

    Music Player Daemon (MPD) running on any Android devices as a Service, on background.

    Simply run aMPD, choose your Music directory and choose if you want to run MPD automatically (at device boot). Then play a variety of sound files remotely via any MPD clients.

    You can use any MPD clients like MPDroid on Android or any other clients on Linux/Windows/Mac.

    Connect your client to MPD using the host and port that is shown when you run aMPD.

    See http://www.musicpd.org/ for more informations about the MPD protocol.

    Get it

    Download apk from last releases, or download it from Google Play Store.

    For basic installation information see the INSTALL file.
